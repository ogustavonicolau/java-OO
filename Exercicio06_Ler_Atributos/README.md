# Exercício 06 — Ler atributos

## Objetivo

Reforçar a ideia de que você pode **ler** atributos de um objeto e usá-los em `System.out.println(...)` (ou em qualquer lógica).

## Arquivos

- `Aluno.java`
- `Principal.java`

## O que fazer

- Leia o enunciado em `Exercicio_06_Ler_Atributos.md`.
- Rode o `Principal.java` e observe a saída (nome e idade).
- Altere os valores atribuídos e rode novamente.
- Responda no `Resposta_Exercicio_06.md` (ou no seu caderno) as perguntas do enunciado.

## Como executar (Windows)

Na raiz do projeto:

```bat
javac Exercicio06_Ler_Atributos\Aluno.java Exercicio06_Ler_Atributos\Principal.java
java Exercicio06_Ler_Atributos.Principal
```
