# Respostas Exercício 1 - Classe
## Atividade 1:

**O que é uma classe?**

R. Uma classe é um molde que é usada para criar objetos, com seus atributos e ou métodos.

**O que representa o arquivo Aluno.java?**

R. Ele representa uma classe e sempre começa com letra maiúscula assim como o nome class Maiscula(nome da classe).

**Por que o arquivo Principal.java existe?**

R. Porque é nesse arquivo que o código será rodado

**O que é um atributo?**

R. São variáveis declaradas dentro de uma classe, que vão representar as características, propriedades e etc.

**O que o atributo Turma representa?**

# Atividade 6:

**Por que Principal.java possui o método main?**

R. Porque o método main é ponto de entrada principal, é a partir dele que o código é iniciado e executado

**Por que Aluno.java não executa sozinho?**

R. Porque é um arquivo sem método main, nele tem somente alguns atributos.

**O que você aprendeu neste exercício?**
